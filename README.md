# Youtube Video Sharing App

## Introduction

Web to post your favorite Youtube embed link Save and History them.

## Prerequisites

Docker Desktop (If win10/win11) or Docker Linux (Ubuntu)
Git git version 2.53.0.windows.2
NPM 11.12.1
NodeJS v25.9.0
Ubuntu Subsystem Win11
VSCode 1.115.0
ruby "3.4.2"
Rails version: 8.0.2
